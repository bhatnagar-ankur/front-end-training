HTML Links Training: Mastering Anchor Tags

Welcome to HTML links fundamentals! This page covers everything you need to know about creating hyperlinks and anchor tags in HTML.

1. Creating Hyperlinks with Anchor Tags

Basic Anchor Tag Structure

<a href="URL">Link Text</a>
This is a basic hyperlink

Complete Link Examples

<a href="https://www.example.com">Visit Example Website</a>
<a href="about.html">About Us</a>
<a href="/products/laptops.html">View Laptops</a>
Visit Example Website
About Us
View Laptops

Anchor Tag Components

  • Opening tag: <a>
  • href attribute: Specifies the destination URL
  • Link text: The clickable text users see
  • Closing tag: </a>

2. Internal vs External Links

External Links (Absolute URLs)

External links point to pages on other websites and require the full URL including the protocol (http:// or https://).

<a href="https://www.google.com">Google Search</a>
<a href="https://github.com">GitHub</a>
<a href="https://developer.mozilla.org">MDN Web Docs</a>
Google Search
GitHub
MDN Web Docs

Internal Links (Relative URLs)

Internal links point to pages within the same website and use relative paths.

<!-- Same directory -->
<a href="contact.html">Contact Page</a>

<!-- Subdirectory -->
<a href="products/shoes.html">Shoes Section</a>

<!-- Parent directory -->
<a href="../index.html">Home</a>

<!-- Root directory -->
<a href="/services/web-design.html">Web Design Services</a>
Contact Page
Shoes Section
Home
Web Design Services

Fragment Links (Same Page Navigation)

<!-- Link to section with id="top" -->
<a href="#top">Go to Top</a>

<!-- Link to section with id="contact" -->
<a href="#contact">Contact Section</a>

<!-- Link to top of current page -->
<a href="#">Back to Top</a>

<!-- Target elements -->
<h2 id="top">Page Header</h2>
<section id="contact">Contact Information</section>
Go to Top
Contact Section
Back to Top
Best Practice: Use external links for other websites, internal links for your own pages, and fragment links for navigation within the same page.

3. Email and Telephone Links

Email Links (mailto)

<!-- Basic email link -->
<a href="mailto:contact@example.com">Send Email</a>

<!-- Email with subject -->
<a href="mailto:support@example.com?subject=Technical Support">Technical Support</a>

<!-- Email with multiple parameters -->
<a href="mailto:info@example.com?subject=Inquiry&body=Hello, I would like to know more about your services.">
    Contact Us
</a>

<!-- Email with CC and BCC -->
<a href="mailto:sales@example.com?cc=manager@example.com&bcc=admin@example.com&subject=Sales Inquiry">
    Sales Team
</a>
Send Email
Technical Support
Contact Us
Sales Team

Telephone Links (tel)

<!-- Basic phone link -->
<a href="tel:+1234567890">Call Us: (123) 456-7890</a>

<!-- International format -->
<a href="tel:+44-20-7946-0958">UK Office: +44 20 7946 0958</a>

<!-- With extension -->
<a href="tel:+1234567890,123">Call Support: (123) 456-7890 ext. 123</a>

<!-- Mobile optimized -->
<a href="tel:+1-555-123-4567">Emergency: (555) 123-4567</a>
Call Us: (123) 456-7890
UK Office: +44 20 7946 0958
Call Support: (123) 456-7890 ext. 123
Emergency: (555) 123-4567

SMS Links

<!-- Basic SMS link -->
<a href="sms:+1234567890">Send SMS</a>

<!-- SMS with pre-filled message -->
<a href="sms:+1234567890?&body=Hello, I'm interested in your services">
    Text Us
</a>
Send SMS
Text Us
Email Parameters:
  • subject - Pre-fills the email subject
  • body - Pre-fills the email body
  • cc - Carbon copy recipients
  • bcc - Blind carbon copy recipients

4. Link Attributes: target, title, rel

Target Attribute

Controls where the linked page opens.

<!-- Opens in new tab/window -->
<a href="https://example.com" target="_blank">External Site (New Tab)</a>

<!-- Opens in same window (default) -->
<a href="page.html" target="_self">Same Window</a>

<!-- Opens in parent frame -->
<a href="content.html" target="_parent">Parent Frame</a>

<!-- Opens in full window -->
<a href="fullpage.html" target="_top">Full Window</a>
External Site (New Tab)
Same Window
Parent Frame
Full Window

Title Attribute

Provides additional information displayed as a tooltip on hover.

<a href="https://github.com" 
   title="Visit GitHub - The world's largest code repository">
   GitHub
</a>

<a href="mailto:support@example.com" 
   title="Send an email to our support team">
   Email Support
</a>

<a href="tel:+1234567890" 
   title="Click to call our office during business hours">
   Call Now
</a>
GitHub
Email Support
Call Now

Rel Attribute

Defines the relationship between the current page and the linked resource.

<!-- External link security -->
<a href="https://external-site.com" 
   target="_blank" 
   rel="noopener noreferrer">
   Secure External Link
</a>

<!-- No follow for untrusted content -->
<a href="https://untrusted-site.com" rel="nofollow">
   Untrusted Link
</a>

<!-- Author information -->
<a href="https://author-portfolio.com" rel="author">
   About the Author
</a>

<!-- License information -->
<a href="license.html" rel="license">
   View License
</a>

<!-- Help documentation -->
<a href="help.html" rel="help">
   Get Help
</a>
Secure External Link
Untrusted Link
About the Author
View License
Get Help

Combined Attributes Example

<a href="https://learn.example.com" 
   target="_blank" 
   rel="noopener noreferrer" 
   title="Learn web development - opens in new tab">
   Learning Resources
</a>
Learning Resources
Security Best Practice: Always use rel="noopener noreferrer" with target="_blank" to prevent security vulnerabilities and protect user privacy.
Accessibility Note: Use the title attribute to provide helpful context, but don't rely on it as the only way to convey important information since it's not accessible to keyboard users.

5. Advanced Link Examples

Image Links

    <img src="logo.png" alt="Company Logo" width="150" height="50">
</a>

Button-styled Links

<a href="signup.html" class="btn btn-primary">Sign Up Now</a>
<a href="tel:+1234567890" class="btn btn-call">Call Us</a>
<a href="mailto:contact@example.com" class="btn btn-email">Email Us</a>
Sign Up Now Call Us Email Us

Navigation Menu

<nav>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="services.html">Services</a></li>
        <li><a href="contact.html">Contact</a></li>
    </ul>
</nav>
  • Home
  • About
  • Services
  • Contact

Quick Reference Guide

Link Type Syntax Use Case
External URL href="https://example.com" Linking to other websites
Internal Page href="about.html" Navigation within your site
Page Section href="#section-id" Jump to page sections
Email href="mailto:email@domain.com" Open email client
Phone href="tel:+1234567890" Initiate phone call
SMS href="sms:+1234567890" Open messaging app
Attribute Values Purpose
target _blank, _self, _parent, _top Where to open the link
title Text string Tooltip on hover
rel noopener, noreferrer, nofollow Relationship and security

Practice Tip: Test your links on different devices and browsers to ensure they work correctly, especially email and phone links!
